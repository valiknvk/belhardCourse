

1. Напишите скрипт, который будет определять доступность указнного сервиса (Например, команда пинг успешно выполняется. Для тестов можно указывать localhost).


2. Напишите скрипт, который будет выводить «Понятно, я получил {значение аргумента}:» в случае передачи ему одного аргумента.
     Если аргументы не были предоставлены, будет выведено «Я не могу работать пустым».
     В случае передечаи более чем одного аргумента выведите "Хэй, мне не нужно столько аргументов!"
     (для реализации использовать конструкцию case)

3. Напишите скрипт, который получает на вход два числа и выводит их сумму
     Убедитесь, в том, что на вход было подано только два числа 
     Протестируйте скрипт, запустив и передав ему два числа в качестве аргументов.
    ( используйте функцию )

4. Необходимо реализовать скрипт, который будет принимть на вход две дерриктории, его вывод должен быть разницей между содержимым двух деррикторий. (можно использовать md5sum)

5. Напишите скрипт, который читайет ввод от пользователя, пока не получит пустую строку
     Для каждой прочитанной строки подсчитайте количество символов в ней и выведите его.
     (Вы должны использовать цикл while)

6. Примечание: предположим, что скрипт выполняется с аргументом

     Напишите скрипт, который будет проверять, является ли заданный аргумент строкой «пицца».
     Если это строка «пицца», выведите «с ананасом?»
     Если это не строка «пицца», выведите «Я хочу пиццу!»

7. Сперва напишите скрипт, который создаёт случайное количество пустых файлов в дирректории и нескольких поддерикториях (можно в ручную задать количество поддиреткорий).Далее создайте скрипт для удаления всех пустых файлов в заданном каталоге (включая вложенные каталоги).

8. Напишите скрипт, который будет парсить json ответ с адреса https://catfact.ninja/breeds и записывать в файл название породы(breeds) и страну происхождения
